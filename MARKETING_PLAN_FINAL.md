## Design System

### Colors
- **Primary Light Blue**: #4FC3F7
- **Primary Dark Blue**: #0B3C5D
- **White**: #FFFFFF
- **Light Grey**: #F5F7FA
- **Dark Grey**: #2D3748
- **Accent Gradient**: Linear gradient from light to dark blue

### Typography (Satoshi)
- **Hero Headline**: Satoshi Black, 64px (desktop), 40px (mobile)
- **Section Headlines**: Satoshi Bold, 48px (desktop), 32px (mobile)
- **Subheadings**: Satoshi Medium, 24px
- **Body Text**: Satoshi Regular, 18px
- **Small Text**: Satoshi Regular, 14px

### Spacing
- Section padding: 120px vertical (desktop), 80px (mobile)
- Container max-width: 1280px
- Grid gap: 32px

### Animations (Minimal & Purposeful)
1. **Fade In on Scroll**: Sections fade in as user scrolls
2. **Hover Scale**: Buttons and cards scale 1.05x on hover
3. **Gradient Shift**: Subtle background gradient animation
4. **Count Up**: Numbers animate on scroll into view
5. **Floating**: Subtle up-down motion on hero mockup
6. **Slide In**: Text slides from side on scroll

**Animation Principles**:
- Duration: 0.3-0.6s
- Easing: ease-in-out
- Trigger: On scroll or hover only
- No auto-playing animations (except subtle background)

### Interactive Elements
1. **Buttons**: Hover glow, scale, color shift
2. **Cards**: Lift with shadow on hover
3. **Icons**: Subtle rotation or scale on hover
4. **Links**: Underline animation on hover
5. **Images**: Parallax scroll effect
6. **Carousels**: Smooth slide transitions

### Tech Vibe Elements
- Subtle grid patterns in backgrounds
- Gradient overlays with tech aesthetic
- Clean, geometric shapes
- Minimal line illustrations
- Modern glassmorphism effects on cards
- Subtle glow effects on interactive elements

## Technical Implementation

### Tech Stack
- React + TypeScript
- Tailwind CSS + shadcn-ui
- Framer Motion (for animations)
- React Intersection Observer (scroll triggers)
- Embla Carousel (for sliders)
- React Countup (for stats)

### Performance
- Lazy load images
- Optimize animations (GPU acceleration)
- Code splitting
- Image optimization (WebP format)
- Minimal bundle size

### Responsive Design
- Mobile-first approach
- Breakpoints: 640px, 768px, 1024px, 1280px
- Touch-friendly interactions
- Optimized for all devices
